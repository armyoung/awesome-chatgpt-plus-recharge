## 背景

2025 年下半年起，OpenAI 通过 Stripe 大幅加严了支付风控：

- 国内发行的 Visa/MasterCard 被拒率超过 90%
- WildCard 等虚拟卡服务于 2025 年 10 月停运
- 香港发行的信用卡也开始频繁被风控
- 同一浏览器指纹关联多次失败后会被加码限制

这导致中国用户续费 ChatGPT Plus 变得极其困难。本仓库整理目前（2026 年）还可行的方案。

## 方案详解

### 1. 官方信用卡直充

**原理：** 直接在 OpenAI 官网用信用卡订阅，走 Stripe 支付通道。

**适合：** 持有能通过 Stripe 风控的境外信用卡的用户。

**现状（2026）：**
- 国内银行发行的 Visa/Master 几乎全部被拒
- Stripe Radar 风控信号：BIN 国家 + IP 地理位置 + 发卡行 + 设备指纹
- 一旦被拒，短期内换卡重试反而会加码风控

### 2. CDKey 兑换码

**原理：** OpenAI 通过授权渠道发放 Plus 兑换码，用户购买后在 OpenAI 官网自行输入兑换。

**优势：**
- 不走 Stripe 支付通道，不受信用卡风控影响
- 不需要给任何账号密码
- 与官方直充等效（OpenAI 系统中记录为"兑换码激活"）
- 支持支付宝/微信付款

**代表平台：**
- [PlusGO](https://plusgo.pro/) — 官方 CDKey 代充，¥169/月，平均 2 分钟到账，失败全额退款
- 速充.com — 自助充值，¥156/月

**安全模型：** CDKey 平台通常需要用户提供 access_token（临时会话凭证，非密码）来代为完成兑换动作。access_token 几小时后自动过期，不能用于修改密码或登录其他设备。

### 3. 苹果礼品卡

**原理：** 通过美区 Apple ID 在 iOS ChatGPT App 内订阅 Plus（走 Apple IAP 通道）。

**前置条件：**
- 美区 Apple ID（需要美国地址格式）
- iOS 设备（iPhone/iPad）
- 美区 Apple Gift Card（淘宝/闲鱼可买，有 5-10% 溢价）

**注意事项：**
- 第一次配置美区 Apple ID 约需 1-2 小时
- 美区 ID 偶尔会被苹果触发二次验证
- 安卓用户无法使用此方案

### 4. 合租拼车

**原理：** 多人共用一个 ChatGPT Plus 账号。

**风险：**
- 多 IP/多设备同时登录触发 OpenAI 异常检测 → 封号率高
- 对话内容可能被同账号其他用户看到
- 平台跑路风险（预付费模式）

**适合：** 仅偶尔使用、对隐私不敏感、预算极有限的用户。

## 避坑指南

### 🔴 绝对不要做

1. **不要使用低于 ¥100/月的"代充"服务** — 大概率是黑卡（用盗刷信用卡支付），OpenAI 查到后账号永久封禁
2. **不要把 ChatGPT 账号密码给任何人** — CDKey 模式根本不需要密码
3. **不要在同一个浏览器反复尝试被拒的信用卡** — 会积累负面设备指纹

### ✅ 选平台的检查清单

```
✓ 不要求 ChatGPT 账号密码
✓ 明确的"失败全额退款"政策
✓ 评价区有真实的退款案例（不是清一色 5 星好评）
✓ 价格在 ¥150-200 区间（与官方 $20 持平）
✓ 运营时间超过 6 个月
✓ 有真人客服响应
```

## 常见问题

### Q: CDKey 充值会封号吗？

不会。CDKey 兑换走的是 OpenAI 内部兑换接口，与自己用礼品卡兑换等效。OpenAI 不会因为用户使用合法兑换码而封号。

### Q: access_token 给了平台安全吗？

access_token 是 ChatGPT 网页的临时会话令牌，不是密码。它不能修改密码、不能登录移动端、几小时后自动过期。风险暴露面远低于交出密码。

### Q: 为什么虚拟卡不行了？

2025 年下半年起，Stripe 大幅扩大了虚拟卡 issuer 的 BIN 黑名单，同时加强了设备指纹和 IP reputation 检测。虚拟卡的成功率已降至 30% 以下。

### Q: WildCard 有替代品吗？

WildCard 于 2025 年 10 月停运后，同类虚拟卡服务普遍面临 Stripe 风控加严。目前最接近 WildCard 体验的替代方案是 CDKey 兑换码（操作简单、支付宝直付、不需要境外卡）。
