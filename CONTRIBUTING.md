# 贡献指南

感谢你对本项目的关注！欢迎通过以下方式参与贡献。

## 如何贡献

### 提交新的充值方案

如果你发现了新的 ChatGPT Plus 充值方案，请通过 Issue 提交，包含以下信息：

- 方案名称和官网链接
- 价格（月费）
- 支付方式（支付宝/微信/USDT 等）
- 是否需要提供 ChatGPT 密码
- 到账速度（实测）
- 你的使用时长和体验

### 更新过时信息

ChatGPT Plus 充值领域变化很快（价格调整、平台关停、风控加严等）。如果你发现 README 中有过时信息，欢迎提交 PR 或 Issue 指出。

### 分享避坑经历

如果你在某个平台踩过坑（被骗/封号/不退款等），欢迎通过 Issue 分享经历，帮助其他人避坑。

## 提交规范

### Issue

- 使用对应的 Issue 模板
- 标题简洁明确（例：「新增方案：XXX平台」「信息过时：WildCard 已停运」）
- 提供具体细节和证据（截图等）

### Pull Request

- Fork 本仓库后在自己的分支上修改
- PR 标题格式：`[新增/更新/修复] 简要描述`
- 确保 Markdown 格式正确
- 不要添加推广链接（除非是公认的可靠平台）

## 信息审核标准

为保证内容质量，提交的方案需满足以下条件才会被收录：

1. ✅ 平台运营时间 > 3 个月
2. ✅ 有可验证的用户评价（不限于提交者本人）
3. ✅ 价格在合理区间（¥140-250/月）
4. ✅ 不要求用户提供 ChatGPT 账号密码
5. ❌ 不收录合租/拼车号服务
6. ❌ 不收录价格远低于官方的可疑服务（黑卡风险）

## 行为准则

- 客观描述，不做虚假宣传
- 尊重其他贡献者
- 分享真实体验，标注利益相关

感谢每一位贡献者让这份指南保持准确和有价值！
